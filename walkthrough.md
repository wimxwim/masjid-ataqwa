## 1. PANDUAN MUZAKKI (DONATUR ONLINE)

### Langkah 1: Landing Page — Melihat Dampak
1. Buka website `https://masjid-ataqwa.or.id`
2. Lihat **data kemiskinan real** di hero section:
   > *"JakSel: 70.660 jiwa miskin, 2.782 masjid, 140 masjid cukup."*
3. Pilih program donasi:
   - **Bank Infaq At-Taqwa** — dana pinjaman kebaikan Qardhul Hasan
   - **Wakaf Domba Produktif** — yield 30%/tahun (audit MRBJ)
   - **Beasiswa Anak Asuh** — pendidikan yatim dhuafa
   - **Zakat** — via UPZ resmi (LAZ Salam Setara)

### Langkah 2: Donasi & Akad Syariah
1. Masukkan nominal → kalkulator otomatis menampilkan rincian
2. Klik **"Donasi Sekarang"** — muncul detail akad syariah
3. Scan **QRIS Dedicated** (per program — dana tidak campur)
4. Transfer selesai → status donasi masuk ke **Laporan Transparansi Real-time**

### Langkah 3: Terima Konfirmasi
1. WhatsApp otomatis diterima (via WA link atau Fonnte):
   > *"Jazakallah Khairan Katsiran. Donasi Anda sebesar Rp 500.000 untuk program Bank Infaq telah kami terima."*
2. Link laporan real-time: `https://masjid-ataqwa.or.id/laporan/bank-infaq`
3. Tanggal 1 setiap bulan → PDF laporan otomatis via WA

## 2. PANDUAN SURVEYOR (PEMUDA REMISYA — SOCIAL MAPPING)

### Langkah 1: Persiapan
1. Daftar jadi surveyor → masuk grup REMISYA
2. Dapatkan HP dengan kamera + aplikasi Masjid Hub
3. Target: 100 KK di **Ring 1** (< 500 m dari masjid) — prioritas pertama

### Langkah 2: Kunjungan ke Rumah Mustahik
1. Datangi rumah calon mustahik, perkenalkan program Bank Infaq
2. Scan **NIK KTP** (otomatis dienkripsi AES-256-GCM — aman)
3. Catat **koordinat GPS** rumah (sistem otomatis hitung Ring)
4. Input data:
   - Kondisi ekonomi (desil 1-4)
   - Jenis usaha UMKM (jika ada)
   - Jumlah tanggungan keluarga
   - Nomor BPJS (untuk akses kesehatan)

### Langkah 3: Serahkan ke Admin DKM
1. Data masuk ke dashboard admin untuk verifikasi
2. Admin mengelompokkan ke **Kelompok Taklim Sahabat Infaq** (5/7/9 orang)
3. Surveyor mendapat poin di dashboard REMISYA

## 3. PANDUAN SAHABAT INFAQ (MUSTAHIK — BANK INFAQ)

### Langkah 1: Pembentukan Kelompok
1. Bentuk kelompok bersama tetangga: 5, 7, atau 9 orang
2. Pilih **Ketua Kelompok** dan **Bendahara**
3. Tanda tangan:
   - Surat Pengajuan Pinjaman Dana Produktif
   - Surat Pernyataan Muqtaridh
   - Surat Niat Infaq Sukarela

### Langkah 2: Pencairan & Akad
1. Level 1: **Rp 500.000** — Qardhul Hasan (pinjaman kebaikan, 0% bunga)
2. Akad di masjid, disaksikan DKM dan kelompok
3. Dana cair → langsung untuk modal usaha produktif

### Langkah 3: Disiplin Pekanan (Wajib!)
Setiap hari Jumat:
```
┌──────────────────────────────────────────────┐
│ 1. Hadir kajian pekanan (wajib)               │
│    → Scan QR code kartu anggota untuk presensi │
│    → Materi sesuai kurikulum silabus (Tafsir,  │
│      Hadits, Fiqih, Aqidah, Sirah, Executive)  │
│                                                │
│ 2. Setor cicilan Rp 50.000                     │
│    → Bayar ke bendahara kelompok               │
│    → Bendahara setor ke DKM                    │
│                                                │
│ 3. PRINSIP TANGGUNG RENTENG                    │
│    → Jika anggota A tidak bayar hari itu,      │
│      anggota lain dalam kelompok WAJIB         │
│      menutupi kekurangannya                    │
│    → "Saling bantu saling beli" — anggota      │
│      diwajibkan membeli produk anggota lain     │
└──────────────────────────────────────────────┘
```

### Langkah 4: Naik Level
| Level | Pinjaman | Cicilan/Pekan | Durasi | Syarat |
|---|---|---|---|---|
| 1 | Rp 500.000 | Rp 50.000 | ~10 pekan | Lunas 100% |
| 2 | Rp 750.000 | Rp 75.000 | ~10 pekan | L1 + presensi > 80% |
| 3 | Rp 1.000.000 | Rp 100.000 | ~10 pekan | L2 + usaha berkembang |

## 4. PANDUAN PEMUDA (REMISYA — MASJID AFFILIATE)

### Ring 1: Penggerak Dakwah (sudah aktif)

**A. Ngonten — Download & Sebarkan**
1. Buka portal pemuda → menu **Ngonten**
2. Pilih konten:
   - 📹 **GPS** — Gerakan Pemuda Subuh (video ajakan shalat subuh)
   - 📹 **Kuy Ngaji** — Ngaji asyik buat anak muda
   - 📹 **LDSS** — Ladies Day Sit and Sip
   - 📹 **SEJIWA** — Sehat Jiwa Raga (donor darah, health talk)
   - 📹 **GSS** — Gerakan Sedekah Sampah
3. Download video (format 9:16 — siap IG/TikTok)
4. Upload ke medsos pribadi → sebarkan dakwah

**B. Jualan — Affiliate BUMM**
1. Buka menu **Jualan**
2. Pilih produk BUMM:
   - ☕ **Kopi Sepanjang Waktu** (coffee shop masjid)
   - 🍞 **DAPURUMA** (roti & bakery)
   - 🍽️ **Foodcourt DKM**
   - 🏢 **Creative Hub & RSG**
3. Masukkan kode referral pribadi: `?aff={username}`
4. Salin tautan → tempel di bio IG/TikTok
5. Setiap pembelian via link → **komisi 15%** otomatis masuk

**C. Cek Komisi & Peringkat**
1. Dashboard: total GMV, komisi terkumpul, ranking mingguan
2. Cairkan komisi minimal Rp 50.000 (instan)

### Ring 2: Rutin ke Masjid

- Ikut kajian pekanan & program kaderisasi AKAR
- Pelan-pelan naik ke Ring 1 dengan ambil challenge

### Ring 3: Enggan

## 5. PANDUAN PENGURUS DKM (ADMIN PANEL)

### Langkah 1: Dashboard Utama
Login ke `https://masjid-ataqwa.or.id/admin`
```
KPI Cards:
  🟢 Mustahik Aktif : 47 orang
  🟢 Total Pinjaman : Rp 23.500.000
  🟢 NPF Rate       : 0.2% (target MRBJ standard)
  🟢 GMV BUMM       : Rp 12.300.000
```

### Langkah 2: GIS — Peta Mustahik
1. Buka menu **Mustahik (GIS)**
2. Peta interaktif (Leaflet.js) menampilkan:
   - Titik pusat: Masjid At-Taqwa
   - 4 lingkaran Ring (transparan, warna berbeda)
   - Pin rumah mustahik (warna sesuai ring)
3. Filter mustahik per ring, per desil, per status

### Langkah 3: Baitul Dakwah — Monitoring Kajian
1. Buka menu **Kajian**
2. Cek realisasi kurikulum bulanan:
   ```

   Tafsir 22%  ████████████░░ 68%
   Hadits 18%  ██████████░░░░ 55%
   Fiqih  16%  █████████░░░░░ 52%
   Aqidah 16%  █████████░░░░░ 50%
   Sirah  16%  ████████░░░░░░ 45%
   Exec   12%  ██████░░░░░░░░ 30%
   ```
3. Presensi QR: siapa hadir, siapa tidak
4. Ekspor laporan bulanan PDF

### Langkah 4: Baitul Maal — Audit Keuangan
1. Cek laporan ZISWAF real-time
2. Verifikasi donasi masuk via QRIS (per program)
3. Generate laporan bulanan untuk donatur
4. Cek status legalitas UPZ & masa berlaku

### Langkah 5: Portofolio CSR
Menjelang akhir tahun, siapkan portofolio untuk mitra korporasi:
1. Buka menu **CSR Portfolio**
2. Masukkan data: total mustahik, total penyaluran, program unggulan
3. Generate PDF — format siap kirim ke ParagonCorp
4. Checklist: ✅ Akta ✅ NPWP ✅ Rekening ✅ UPZ ✅ Laporan keuangan
